<h3>Features</h3>

  1. user authentication in django<br>
  2. update the task<br>
  3. delete a task<br>
  4. edit a task<br>
  5. set priority (1 to 5)<br>
  6. Task Filters<br>
  7. due date<br>
  8. search functionality
